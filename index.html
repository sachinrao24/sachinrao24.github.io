Hello! I'm Sachin Rao.

I am an earnest senior year Information Science Engineering undergraduate at the RNS Institute of Technology with experience in data science, machine learning and an intent to contribute to society.

I'm also an aspiring researcher, primarily interested in NLP, with an upcoming publication in IEEE Xplore for my paper titled "N-ary Relation Extraction using Web-scraped Data".

Projects

  • N-ary relation extraction using web-scraped data

    • Scraped news articles from Wikipedia using BeautifulSoup.
    • Used GloVe embeddings and performed relation extraction for n entities where n>2 using a Bi-LSTM model and a multi-task learning framework.
    • Implemented using Keras and NLTK.
    • Paper accepted in the ICAECIS-2023 conference and will be further published in the IEEE Xplore journal.

  • End-to-end ASR system

    GitHub

    • Trained a character-based RNN language model and a CNN-RNN acoustic model for speech recognition on the Russian Open STT dataset.
    • Performed fine-tuning with Reinforcement Learning and RNN-T loss.
    • Implemented using PyTorch.

  • Coffee shop chatbot

    GitHub

    • Created a chatbot for a coffee shop to answer queries regarding available items, payments, shipping, etc.
    • Performed stemming and tokenization before creating a bag of words which serves as the input to a Feed Forward neural net.
    • The input was classified into tags using the softmax function and the chatbot's replies were determined by the type of tag.
    • Implemented using NLTK and PyTorch.
